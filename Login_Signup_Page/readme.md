 Signup & Login Page

A responsive Signup and Login Page built using HTML, CSS, and JavaScript during my TechnoHacks Virtual Internship. The project features a clean user interface for user registration and login.

 Features
- User Signup and Login forms
- Responsive and modern UI
- Form validation using JavaScript

 Technologies Used
- HTML
- CSS
- JavaScript

 How to Run
1. Download or clone the repository.
2. Open the HTML file in your web browser.
3. Use the signup and login forms to interact with the application.
